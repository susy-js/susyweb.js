=================================
susyweb.js - Sophon JavaScript API
=================================

susyweb.js is a collection of libraries which allow you to interact with a local or remote Sophon node,
using an HTTP, WebSocket or IPC connection.

The following documentation will guide you through :ref:`installing and running susyweb.js <adding-susyweb>`,
as well as providing a :ref:`API reference documentation <#id1>` with examples.
